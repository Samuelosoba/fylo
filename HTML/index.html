Introduction to web development

Web development involves building web applications

It is broadly divided into;

  1. Frontend engineering
  2. Backend engineering

Some of the skills needed as a fullstack engineer are

  • Critical thinking
  • Problem solving skills
  • Time management
  • communication skills

Some of the technology used are;

  • HTML
  • CSS
  • Tailwind
  • Javascript
  • React
